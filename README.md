# Employee Attendance System in C

## Description
This project is a simple **Employee Attendance System** built using the C programming language. It allows organizations to record and manage employee attendance efficiently. The system stores attendance data, enables employees to mark their presence, and provides an overview of attendance records.

## Features
- Add new employees to the system
- Mark employee attendance
- View attendance records
